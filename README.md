# Installing-Servers
NGNIX, APACHE, MYSQL, PHP ....

## Install Apache2.

    sudo apt install apache2
    sudo /etc/init.d/apache2 status
    sudo /etc/init.d/apache2 start
    sudo /etc/init.d/apache2 stop
    sudo /etc/init.d/apache2 restart

## Install Nginx.

    sudo apt install nginx
